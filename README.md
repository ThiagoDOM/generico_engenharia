## Sobre o projeto

Sistema desenvolvido para oportunidade de emprego

## Instalação

Após clonar o repositório, execute esse comando:
```
composer install
```

- Crie o arquivo .env usando como base o aquivo .env.example. Ou adicione "FILESYSTEM_DRIVER=public" ao seu .env
